\# 🧠 StudyMate - AI Powered Study Notes App



StudyMate is a full-stack study notes application built with React, Express, and MongoDB.



\## Completed Features

\- React frontend

\- Display study notes from API

\- Express REST API

\- MongoDB Atlas database integration

\- CORS enabled API



\## Tech Stack

Frontend:

\- React + Vite



Backend:

\- Node.js

\- Express.js

\- MongoDB

\- Mongoose
